## English

### What is Browser Agent?

A Chrome extension that helps you automate browser tasks using AI.

### Features

- Chat with AI about web pages
- Automate browser tasks
- Extract content from websites
- **File Management System**
  - Preview and edit CSV files
  - View HTML files
  - Search and filter files
  - Download and manage generated files

### Demo

<video src="20260223-0231-15.6177277.mp4" controls width="100%"></video>

### Installation

1. Download this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the `chrome-extension` folder

### Usage

1. Click the extension icon
2. Configure your API key in settings
3. Start chatting or give it a task
4. **Manage Files**: Switch to "Files" tab to view and edit generated files

### English

#### Overview
The extension now includes a complete file management system for viewing, editing, and managing generated CSV and HTML files.

#### Features
- **File List**: View all generated files with type, size, and creation time
- **CSV Preview**: View and edit data in table format
- **HTML Preview**: Render HTML content or view source code
- **File Operations**: Copy, download, delete, and rename files
- **Search & Filter**: Find files by name or filter by type

#### How to Use
1. Click the extension icon
2. Switch to the "Files" tab
3. Click "Preview" to view a file
4. Edit CSV cells directly and click "Save Changes"
5. Use search box or filter buttons to find files
